# AI Campus Assistant

This project implements an AI Campus Assistant in three levels:

## Level 1: Basic AI Assistant
- Simple chatbot using a pre-trained language model.

## Level 2: Knowledge-Aware Assistant (RAG)
- Uses campus documents to retrieve relevant information.

## Level 3: Agentic Assistant
- Decides whether to answer directly, retrieve documents, or ask clarifying questions.

This assistant helps students navigate campus life efficiently.
